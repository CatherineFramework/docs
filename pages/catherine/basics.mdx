Catherine is a simple framework, so all commands are fairly easy to activate.

Here's a snippet from Catherine displaying the help menu:
```
start_server: Start a Rust server

scan_ports: Scan for open local ports

search_exploit: Search ExploitDB for an available exploit to review

set_module: Set one of Catherine's modules

set_decode: Decode an encoded message using one of our provided methods

view_modules: Currently installed modules

sys_info: Print local system information to stdout

set_windows_module: Allows you to use a module created for Windows

help: Help menu

version: Version info for Catherine framework

exit: Exit Catherine framework
```

You can boot the framework up by running:
```bash
./catherine-beta
```

Once the framework has completed the boot process, you can then run any of the commands above. Our interactive shell will walk you through everything!
